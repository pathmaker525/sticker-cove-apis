### Make Stickers

- Orders database which will hold: <br/>

  Customer Details

  - First Name (Mandatory)
  - Last Name (Mandatory)
  - Email (Mandatory)
  - Phone Number
  - Shipping Address

  Order Details

  - Image for Customer upload
  - Sticker Type
  - Sticker Quantity
  - Sticker Price
  - Proof Image for Admin upload
  - Order Status (open order/ delivering / done)
  - Shipping Label from Stamps.com API
